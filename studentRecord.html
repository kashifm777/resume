Student Record Management System

Student Result Management System project is written in Python. This is a simple console based system which is very easy to understand and use. Talking about the system, it contains basic functions which include Add students, view students, search students and remove the student. In this mini project, there is no such login system. This means he/she can use all those available features easily without any restriction. It is too easy to use, he/she can check the total student’s record.

While adding the students, the user only has to enter name then the system adds the record and displays to the user. And the user can view all these students lists from the view section. In this Student Record Management, the user can also search for student’s name in order to know whether the student’s record exists in the system or not. This simple console based Student Record Management system provides the simplest management of student’s list. In short, this projects mainly focus on CRUD. There is a database connection and external file used in this mini project to save user’s data permanently.
