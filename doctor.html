Online Doctor Appointment System

Online Doctor Appointment System is developed in C# .NET Framework. It is used to mange the patient online appointment records. This automated system is specially made of C# and SQL Server database. The main goal of this system is to minimize the time during manual appoint record . It is so easy to use that even a newbie can use the features with ease.
