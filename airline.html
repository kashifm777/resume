Airline Management System

Airline Ticket Reservation is Web based Application in PHP. It is used to manage the ticket record online. An automated system that is specially made of PHP and MySQL database.

It has following modules:

  1. Signup As customer
  2. Login
  3. Book Ticket
  4. Print Ticket
  5. Check PNR

Admin has following rights in the system:

  • List Flights
  • Arrange Flights
  • Add Flights
  • Schedule Flights
  • Check Booked Tickets
